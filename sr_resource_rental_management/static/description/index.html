This odoo application can be used by any rental industry like Resource Rental, Equipment Rental, Service Rental, Machinery Rental, Fleet Rental and Product Rental. This application covers all the general functions of any rental industry. In this application, you can schedule your rental order, reserve rental products, pickup and return rental orders. You can also configure remainder email for pickup and return rental products.
